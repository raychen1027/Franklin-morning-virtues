# 富兰克林晨间准则技能

## 功能概述

帮助用户通过每日晨间提醒来修习富兰克林的13条道德准则。技能核心功能：
1. 根据当前 ISO 周数，确定本周应专注的准则（第 1 周 = 准则 1，第 14 周 = 准则 1，以此类推）
2. 为用户设置每天早上 6 点的自动提醒（通过 CronCreate）
3. 生成包含三部分内容的 formatted message：
   - 准则原文（中文名 + 英文名 + 英文原文）
   - 古今中外相关名人事例
   - 针对 35-50 岁职业经理人的今日行动建议

## 触发条件

只要用户话语中涉及以下任一概念，立即触发本技能：
- 富兰克林 / Franklin / 13 条 / 十三条 / virtues
- 晨间提醒 / 早上提醒 / 每日提醒 / morning reminder
- 设置定时 / 定时提醒 / 每日早课
- 生成今天的准则 / 今日准则 / 今天该练哪一条

## 指令流程

### 步骤 1：判断用户意图

- 如果用户在**设置提醒或定时任务**（关键词：设置、设定、安排、定时、每天早上、晨间提醒、提醒我），跳到 **步骤 2A**
- 如果用户在**请求生成今日内容**（关键词：今天、今日、生成、输出、给我看看），或者本技能由 cron 任务触发，跳到 **步骤 2B**

### 步骤 2A：设置每日 6 点定时提醒

使用 `CronCreate` 工具创建定时任务：
- **cron**: `0 6 * * *`
- **prompt**: `生成今日富兰克林晨间准则提醒`
- **durable**: `true`
- **recurring**: `true`

完成后，报告用户提醒已设置成功，然后**继续执行步骤 2B**，展示今日的准则内容。

### 步骤 2B：生成今日富兰克林准则内容

1. **运行脚本获取当前准则**：
   使用 `Bash` 工具运行 bundled 脚本。优先尝试 `python3`，如失败再尝试 `python`：
   ```bash
   python3 <skill_path>/scripts/virtue_selector.py
   ```
   如果 Bash 被完全拒绝，则手动计算：
   - 今天是 ISO 第 N 周（使用 `datetime.now().isocalendar().week` 的逻辑）
   - virtue_index = (N - 1) % 13  → 这会得到 0-12 的索引
   - 对应准则：索引 0=节制, 1=缄默, 2=秩序, 3=决心, 4=节俭, 5=勤奋, 6=真诚, 7=正义, 8=中庸, 9=清洁, 10=宁静, 11=节欲, 12=谦逊

   **注意**：ISO 第 13 周 → 索引 12 → 谦逊 (HUMILITY)

2. 解析 JSON（或手动计算结果），获取：ISO 周数、准则编号、中文名、英文名、英文原文。
3. 依据下方格式要求，生成最终文本输出。

**重要**：ISO 周数在输出标题中**不要加空格**，格式为 `第13周` 而非 `第 13 周`。

## 输出格式规范

必须严格使用以下三段式结构。整体风格**简洁、优雅、不 preachy**，便于快速阅读与执行。

```
🌅 富兰克林晨间准则 | 第 {{iso_week}} 周

━━━━━━━━━━━━━━━━━━━━
一、本周准则
━━━━━━━━━━━━━━━━━━━━
{{virtue_cn}} | {{virtue_en}}

"{{quote}}"

简要阐释：用 1-2 句话说明这条准则在现代职场中的核心含义。

━━━━━━━━━━━━━━━━━━━━
二、名将通鉴
━━━━━━━━━━━━━━━━━━━━
选取一位古今中外真实历史人物（中外皆可，如曾国藩、罗斯福、稻盛和夫、康德、华盛顿、张居正等），简述其与此准则相契合的一个具体事例。控制在 80 字以内，只讲事实，不加评论。

━━━━━━━━━━━━━━━━━━━━
三、今日行动
━━━━━━━━━━━━━━━━━━━━
针对一位 35-50 岁的职业经理人（事业上升期，肩负团队与家庭责任），给出一条具体、可执行、不空洞的今日行动建议。建议要小到能在今天完成，大到对工作生活有实质影响。控制在 50 字以内。
```

### 格式约束
- 全文以中文为主，仅准则部分保留英文原文
- 历史事例必须真实、具体，避免泛泛而谈
- 行动建议必须针对性强、当日可执行
- 总长度控制在 300 中文字以内（不含标点和空行）
- 语气：平和、理性、略带古典质感
